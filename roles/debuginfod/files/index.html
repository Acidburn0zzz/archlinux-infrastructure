Debuginfod service - Arch Linux

This is the debuginfod service for Arch Linux. For more information about it, please refer to https://wiki.archlinux.org/title/Debuginfod.

The following environment variable must be set in your shell to use it:

DEBUGINFOD_URLS="https://debuginfod.archlinux.org"

A dashboard can be found on https://dashboards.archlinux.org/.

List of currently available packages can be found on here.
